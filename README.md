# Chat-App
Chat app with rooms

## Custom message color
You can use `${number}` in front of your message to change its color - "$5Hello"

## Getting started
### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/JstFlip/Chat-App.git
   ```
2. Install NPM packages
   ```sh
   yarn
   ```
3. Run in `development` or `normal` mode
   ```sh
   yarn dev
   ```
   ```sh
   yarn start
   ```
